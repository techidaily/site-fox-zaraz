## Files Content

Displays the content of the INI files you added in the INI files section. The files are merged into a single file.

**Name** \- Set the name of the INI file that contains the INI content of the merged files.

**Keep formatted references** \- Enable/disable the ability to escape formatted references.

## Component

**Existing** \- Select a component from the drop-down to which you want to import the INI values.

**New** \- Set the name of a New component, and import the INI values to it.
